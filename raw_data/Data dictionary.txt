# Overview

This project makes a plot of participant ASRS score against average microsaccade count

Data from an experiment run by Tom and Elliot. Details here:

Hampsey, E., Overton, P.G., Stafford, T. (2019). Microsaccade rate as a measure of drug response. 12(6). https://doi.org/10.16910/jemr.12.6.12


# What is here

README.txt - project description. This file
make_graph.py - script for doing the analisis, in Python
datawrangle_exercises.R - comments only script for what we want to do in R
datawrangle_exercises_complete.R - script for doing similar 
\data - folder with the data in
\outputs - folder with the graph output in
\docs - folder containing a completed R script
